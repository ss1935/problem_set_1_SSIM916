# problem_set_1_SSIM916
Repository for 740098762's problem set 1 (SSIM916 Machine Learning for Social Data Science)

**Raw dataset (csv file):**
- Telco's customer churn (WA_Fn-UseC_-Telco-Customer-Churn)


**Codes included in this repository (all included within 1 jupyter notebook):**
- preparation code for the dataset (DATASET PREPARATION CODE)
- fool-proof code for checking the prepared dataset
- EDA of the dataset + 2 visualisations used
- logistics regression and random forest models + model summary + visualisations
- visualisation of variables with the most affect towards churn
- table comparison of each models' results


**How to run:**
- adjust the file path in the 'DATASET PREPARATION CODE' according to the path of the 'WA_Fn-UseC_-Telco-Customer-Churn' dataset
- run the 'DATASET PREPARATION CODE' code first before running the rest of the codes
